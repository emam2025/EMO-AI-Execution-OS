Bugs Fixed During Implementation
---------------------------------
1. Artifact import path: from artifacts.design.phase_l.models.02_memory_and_context_models
   was syntactically invalid (file starts with digit). Fixed by creating core/memory/models.py
   and updating 3 files to import from there.
2. TokenBudget scaling: When max_tokens < 4096, __post_init__ assertions failed because
   default reservations summed to 4096. Added TokenBudget.scaled(max_tokens) classmethod
   for proportional budget scaling.
3. MemoryStateMachine: No transition from IDLE to EXECUTION_COMPLETE state. Added T0
   transition to handle the initial state transition.
4. MemoryHierarchy.get_context_window(): Was missing SHA-256 hash computation. Added
   context_hash field using hashlib.sha256 over sorted JSON of window data.
5. Test async support: Tests needed @pytest.mark.anyio for async def methods.
